# Prueba E2E
Autor:  Angie Alvarado

Prueba de automatización creada usando Karate Framework

## Prerrequisitos

La herramientas utilizadas para la creación de este proyecto son:

- IntelliJ IDEA 
- Java JDK 17 
- Mave 3.9.1




---

## Plugins necesarios en IntelliJ

Se recomienda instalar los siguientes plugins para una mejor experiencia de desarrollo:

- Gherkin
- Cucumber for Java




---

## Ejecución de pruebas

- Ejecutar la prueba desde el archivo PetsTest
- Para ejecutar los endpoints de manera individual ejecutarlo desde PetsRunner

### Ejecutar todos los tests desde consola

 ```
mvn clean test

```
